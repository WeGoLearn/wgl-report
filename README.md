# JS REPORT 3

## Install

```
git clone https://github.com/Popety/popety-io-jsreport.git
cd popety-io-jsreport
yarn install
```

## Start

```
yarn start
```

Open: http://localhost:5488/

## Deploy

```
git add .
git commit -m '[---] ---'
git push origin main
```

## Restart

```
pm2 restart jsreport
```
